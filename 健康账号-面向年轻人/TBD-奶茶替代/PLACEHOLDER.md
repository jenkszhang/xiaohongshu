# 候选选题：奶茶替代品

## 标题预案
- `奶茶戒不住？3 杯你可以留着`（14/20）
- `奶茶党的抗炎替代清单`（11/20）
- `戒奶茶不痛苦：3 杯顶得上`（13/20）

## 核心点（本次讨论提炼）

雷区 3 杯：
- 奶盖茶 / 果茶 / 粉末加奶（糖 + 反式脂肪 + 植脂末三连）
- 大杯多糖（一杯 ≈ 8-12 颗方糖）
- 隐藏雷：「低糖」选项实际仍含 ≈ 4-6 颗方糖

可以留着的 3 杯替代：
1. 冷萃 / 美式咖啡（黑或加 1 勺无糖奶）— 接近拿铁口感但无糖
2. 无糖气泡水 + 鲜柠檬 / 黄瓜片 / 薄荷
3. 冷泡绿茶 + 蜂蜜一小勺（看蜂蜜量）

奶茶解馋的「降级版」：
- 1/4 糖 + 选低脂奶替代奶盖
- 选大叶茶基底而非粉末
- 不要珍珠 / 椰果 / 布丁等高糖配料

## 实拍建议
- 自己拍三种替代杯子并排照
- 一杯奶茶旁边一堆方糖对照
- 便利店货架冷藏柜挑选场景
