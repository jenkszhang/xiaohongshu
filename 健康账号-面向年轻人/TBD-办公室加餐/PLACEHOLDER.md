# 候选选题：办公室抗炎加餐

## 标题预案
- `下午饿到才踩实：7 类拒绝发炎零食`（18/20）
- `办公室加餐党 · 7 样我抽屉常备`（15/20）
- `下午 3 点饿？这 7 样比饼干强 10 倍`（16/20）

## 核心点（本次讨论提炼）

7 样抽屉常备：
1. 坚果（核桃 / 杏仁 / 腰果）
2. 黑巧克力 ＞ 72%
3. 黑豆 / 鹰嘴豆烤
4. 鲜果（榴莲、葡萄当心，蓝莓 / 苹果可）
5. 希腊酸奶 + 莓果
6. 原味燕麦棒 / 黄豆面包片
7. 海苔 / 紫菜

避雷 4 样：
- 饼干 / 曲奇 / 蛋黄派
- 油炸薯片 / 锅巴
- 奶茶 / 含糖饮料
- 网红代餐棒（看配料表有反式脂肪）

## 实拍建议
- 办公抽屉俯拍（坚果罐 / 巧克力盒 / 燕麦棒）
- 装着希腊酸奶+莓果的玻璃杯
- 一颗黑巧克力的特写
- 真实的「3 点准时饿了」工位照
