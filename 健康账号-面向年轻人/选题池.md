## 互动型待选（2026-05-23 第二轮选题对话产出）

### A · 健康智商税暴露
- **类型**: 暴露/雷型
- **标题预案**: 「30 岁前最后悔买的 7 个健康智商税，我中了 4 个」
- **评论区话头**: 你中了哪个？/ 还有哪些补充？/ 最后悔哪个？
- **封面骨架**: 物体网格 6-7 格（防蓝光镜 / 筋膜枪 / 酵素 / 护肝片 / 胶原蛋白 / 褪黑素 / 燃脂咖啡）+ 大红叉
- **主长尾词候选**: 健康智商税 / 智商税产品 / 上班族智商税
- **⚠️ 注意**: 只聊品类，不点品牌名

### B · 身体先垮部位投票
- **类型**: 投票/共鸣型
- **标题预案**: 「30 岁前打工人，身体最先垮的 5 个部位，你中了哪个」
- **评论区话头**: 我才 25 就 X / 我是颈先垮 / 晒自己最先疼的部位
- **封面骨架**: 人体图标注 5 个热区（颈 / 腰 / 眼 / 肠胃 / 膝）
- **主长尾词候选**: 打工人通病 / 上班族身体 / 30岁身体

### C · 早起 vs 熬夜站队
- **类型**: 站队型
- **标题预案**: 「早 5 点起 vs 凌晨 1 点睡，哪派身体更好」
- **评论区话头**: 我是熬派 / 我是晨型人 / 晒作息+身体状况
- **封面骨架**: 左右对比（早起人 vs 熬夜人状态）
- **主长尾词候选**: 早起的人 / 熬夜的人 / 晨型人 / 夜猫子
- **⚠️ 注意**: 作息线密度偏高（已发咖啡时机、皮质醇），延后再发

### D · 亚健康自测打分
- **类型**: 自测型
- **标题预案**: 「打工人亚健康 10 项自测：中 5 项该体检」
- **评论区话头**: 我中了 X 项 / 同事都中了 / 我以为只有我
- **封面骨架**: 清单格 10 项打勾
- **主长尾词候选**: 亚健康自测 / 打工人亚健康 / 亚健康症状
