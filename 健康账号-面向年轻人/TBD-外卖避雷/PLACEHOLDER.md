# 候选选题：外卖踩雷

## 标题预案
- `5 类常点外卖，可能一直在让你发炎`（17/20）
- `天天点外卖？这 5 类正在悄悄发炎`（15/20）
- `打工人外卖踩雷清单 · 抗炎党版`（14/20）

## 核心点（本次讨论提炼）

入坑外卖（5 雷区）：
- 烤饭 / 麻辣 / 水煮肉 / 烤冷面 / 重口味快餐 → 高糖 + 高 omega-6 油 + 高盐
- 油炸主食（炸鸡 / 薯条 / 油条配品）
- 含糖饮料默认搭配
- 加工肉类配菜（火腿 / 香肠 / 肉肠）
- 重酱料 / 重芡汁（糖 + 盐 + 反式脂肪）

优选外卖（3 类）：
- 蔬果 ≥ 3 类 + 一蛋白：日式定食 / 蔓越鸡胸沙拉 / 越南河粉减酱
- 清淡药膳 / 清蒸鱼 / 鸡胸饭
- 茶 / 黑咖 替代含糖饮料

3 步点单法：
- 凑足蔬菜（一半盘子绿）
- 拒绝油炸 + 含糖饮料
- 减半油醒（让餐厅少放油）

## 实拍建议
- 外卖盒打开俯拍（一份配菜对比"踩雷" vs "优选"）
- 蔓越鸡胸沙拉一盘
- 日式定食盘
- 自己拍的外卖小票（不显示个人信息那种）
