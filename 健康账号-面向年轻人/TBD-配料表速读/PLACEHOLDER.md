# 候选选题：配料表速读法

## 标题预案
- `3 分钟学会看配料表，别再踩雷`（16/20）
- `配料表前 3 位出现这些，放下别买`（17/20）
- `打工人速读配料表：3 个红字直接拒绝`（19/20）

## 核心点（本次讨论提炼）

3 红字直接拒绝（前 3 位出现）：
- 「糖 / 果葡糖浆 / 玉米糖浆 / 麦芽糖浆」=糖
- 「氢化植物油 / 起酥油 / 植脂末 / 代可可脂」= 反式脂肪
- 「人造黄油 / 人造奶油」= 同上

谨慎成分（不必直接拒，但记一下）：
- 「植物油」（不写明哪种）→ 大概率玉米/大豆/葵花籽油，promote 炎症
- E 代号：部分人造增味剂 / 防腐剂可能加重炎症
- 「精制 X」（精制白糖 / 精制油）

3 个隐藏陷阱：
- 包装上写「0 糖」但配料表有「糖醇」= 其实有糖
- 「天然成分 X%」剩 X% 是什么没写
- 「低脂」常常意味着高糖

## 实拍建议
- 超市货架手机抓拍（任一零食包装配料表特写）
- 用手指点配料表前 3 位的特写
- 抽屉里几个产品对比照（一目了然差异）
