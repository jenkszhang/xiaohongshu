# 午觉睡完更累？踩了这 3 个雷

## 正文（约 760 字，已去 markdown 格式，可直接复制粘贴到小红书发布器）

这周中午午觉踩雷了，下午3点又困到怀疑人生。趴桌睡满1小时，醒来整个人飘着，眼皮重得像贴了胶带，下午开会接不上同事的话，我突然觉得这事我一直在做错。前两周还在用猛灌冷萃硬撑，结果心慌+更困，全是反效果。我之前一度以为是缺铁，特地跑去抽了血也没看出毛病，折腾一圈才回到午睡这件事本身。

先说时长。睡10-20分钟只进浅睡，醒来反而清醒；过了30分钟开始进深睡，被闹钟拉醒就会陷在「醒不过来」的状态，那种昏沉起码再要30分钟才散。我现在闹钟卡25分钟（包括5分钟入睡时间），亲测不再头晕。上周三特意试了反过来，12:30一睡到13:30整1小时，醒来花了快40分钟才能正常开会，下午基本废了。

姿势这块我以前真没意识到。趴桌睡眼球被压（醒来视力糊一阵），胃被压（午饭刚下肚反酸），颈椎歪一上午。我去年试了两种U型枕都不行：海绵款太软压脖子，记忆棉款又太热出汗。后来换成带椅背勾的硬款，靠住后仰110°才舒服。公司有沙发就直接平躺，没沙发就靠椅背半躺。那个硬款U型枕现在常驻办公桌抽屉，会议密的午休就靠住打个盹，虽然不深但比趴桌强太多。

睡晚也是坑。之前忙起来经常拖到下午2点半才眯一下，这种「晚午觉」会偷走晚上的困意，11点不困、12点更不困，连环坑。稳的时间窗是12:30-13:30睡完，过了2点就别再补了。昨天差点又拖到3点，看了眼时间果断没睡，下午是难熬一点，但晚上10点就困了，比白天补睡划算。

午饭后别立刻趴。我之前吃完就睡反酸了好几次，吃辣的那种反酸更严重。后来固定隔10-15分钟、起来走两步再睡，胃舒服很多。茶水间走一圈、洗个杯子、刷会儿手机都行，就是别一吃完倒头。我们办公室茶水间常年有人围着热水壶聊几句，慢悠悠晃一圈就够10分钟。

调整下来两周，下午能多专注两个小时。午觉这件事真没那么简单，"困了就睡"的逻辑反而把人耗得更狠。还有个意外收获：晚上入睡也快了，不用躺着滑手机1小时才有困意；对下午茶美式的依赖也淡了，以前下午没杯美式像缺了什么，现在偶尔忘了点都没事。
