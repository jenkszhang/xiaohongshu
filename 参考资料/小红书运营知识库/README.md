## ⚠️ 重要：这是「字典」不是「skill」

这个目录**不**会被 Claude Code 自动加载为 skill。它只是知识参考资料，AI 不会主动按关键词触发——避免跟我们工程内 `.claude/skills/xhs-*` 的工作流 pipeline 抢触发。

**使用方式**：
- AI 或用户**主动 `Read`** 某一篇 SKILL.md 作参考
- 跟我们 `.claude/skills/xhs-*` 工作流冲突时，**以工程内 xhs-* 为准**（永久 memory 红线 + 5 处布点 + 纯文本 + 4-6 全 AI 图 + chrome-devtools 只调研）

## 跟工程内 xhs-* skill 冲突时怎么办

**工程内 xhs-* 是工作流，外部库是知识科普**，遇到冲突一律工程内为准。常见冲突点：

| 冲突点 | 外部库说法 | 工程内 xhs-* 说法（以此为准）|
|---|---|---|
| 发布方式 | 部分 skill 暗示自动化 | chrome-devtools 只用于调研，发布手动（[[feedback-xhs-tools-scope]]）|
| 封面图 | 默认 Canva / 醒图 | liclick AI 出图，4-6 张全 AI（[[feedback-xhs-image-count-no-handshot]]）|
| 正文格式 | markdown | 纯文本 + 末行 inline tag（[[xhs-copy]] §6）|
| tag | 5-10 个为宜 | 至少 2 个含主长尾词（[[xhs-copy]] §5）|
| 关键词布点 | 标题 + 内容 + 标签 | **5 处**：标题前 10 字 / 首句 / 中段 / tag / 首评（[[xhs-copy]] §5）|
