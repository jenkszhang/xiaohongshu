# 账号定位 · 上班族-抗炎饮食

## 一句话定位

**专为 25-32 都市上班女做的抗炎饮食实操号**——只讲「上班族场景下怎么吃 / 吃什么 / 避什么」，不讲理论、不讲做菜教程。

---

## 受众画像

**主体受众**
- 性别：女
- 年龄：25-32
- 城市：一线 + 新一线（北上广深杭成都）
- 职业：码农、运营、设计、产品、市场、HR 等坐办公室脑力工作
- 月收入：1-3w
- 婚育：未婚 / 已婚未育为主，少数已育

**生活状态**
- 早晚通勤总时长 ≥ 1.5h
- 工作日午餐外卖比例 ≥ 50%（很多人 70%+）
- 久坐 8h+，缺乏运动
- 自感「亚健康」但不愿大动作改变（不爱做饭、不爱去健身房、不爱看医生）
- 愿意为「低操作成本」的小改变付费 / 试错

**痛点（这些词在标题 / 评论里高频出现）**
- 长痘 / 暗黄 / 皮肤敏感
- 总累 / 总疲劳 / 没精神
- 肠胃胀气 / 便秘
- 过敏增多
- 易胖难瘦 / 腰腹脂肪堆积

## 三个内容边界

### ✅ 做的事

1. **场景化抗炎**：「外卖怎么挑」「便利店早餐怎么选」「下午茶怎么吃不踩雷」
2. **可执行清单**：「这 5 种调味油该常备」「易胀气 / 易长痘 / 易疲惫 该忌口的食物」
3. **真实记录**：「我跟拍 7 天午餐」「换油后皮肤的变化」（亲身实验型）

### ❌ 不做的事

1. **不讲抗炎理论**：不解释「细胞因子」「氧化应激」这种学术原理（留给营养学硕士同行做）
2. **不讲做菜教程**：不出「葱烧鸡腿菇 step1 step2」这种菜谱（留给美食号）
3. **不开 1v1 咨询**：永远以「打工人 vs 打工人分享」自定位，不扮专家

## 内容比例配额（控制不偏题）

每 10 篇笔记的预设结构：
- 5 篇：场景实操（外卖 / 便利店 / 早餐 / 加班餐）
- 3 篇：清单对比（抗炎食物 vs 促炎食物 / 油的选择 / 茶饮选择）
- 1 篇：跟拍记录（7 天午餐 / 换油 / 戒奶茶）
- 1 篇：社群互动（投票 / 问卷 / Q&A）

**互不打架的策略**：

候选方向（3 套，待用户选定，详见 `账号视觉/候选/`）：
